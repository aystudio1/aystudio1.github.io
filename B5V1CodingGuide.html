Make a Window:

A2Y.Create.Window ExampleWindow = new A2Y.Create.Window("Example Window", 400, 400, true, null, 3);

Make a Button:

A2Y.Create.WindowButton button = new A2Y.Create.WindowButton("Example Button", ExampleWindow.window);

Make text in console:

new A2Y.Functions.ConsoleMessage("Example Message!");

Get Date/Time:

Date:

new A2Y.getOSInfo.date();

Time:

new A2Y.getOSInfo.time();

Close a Application:

new A2Y.Functions.CloseApplication(1);
